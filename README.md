**Built an Image Caption Generator using Generative AI**

I recently developed an AI application that automatically generates captions for images using deep learning models.

🔹 Used the BLIP (Bootstrapping Language–Image Pre-training) model for image captioning
🔹 Implemented the solution using Python and the Hugging Face Transformers library
🔹 Built an interactive interface using Gradio
🔹 Supports both local image uploads and image URLs

This project demonstrates how Vision-Language Models can understand visual content and generate meaningful textual descriptions.

Tech Stack: Python | Hugging Face Transformers | BLIP | Gradio | PyTorch
